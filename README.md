**My Portfolio**

A personal portfolio website built with React and Vite to showcase projects, skills, and professional experience.

This repository contains the source code for a responsive and modern portfolio that can be deployed on GitHub Pages, Vercel, Netlify, or any hosting service.

**Tech Stack :**

This project uses the following technologies:

  React — Frontend UI library

  Vite — Build tool for faster development and production builds

  HTML, CSS, JavaScript — Core web technologies

**Features :**

  -Modern UI built with React

  -Fast development experience with Vite

  -Mobile responsive

  -Easy to customize

  -Ready for deployment
